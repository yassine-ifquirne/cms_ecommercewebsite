# MyStore - Clothing Store Website This is a clothing store website built using **WordPress**, **WooCommerce**, and the **Astra** theme. The website is powered by a local database (`mystore_db`) on XAMPP and can be run on a local server environment. ## Table of Contents 1. [Installation Instructions](#installation-instructions) 2. [Administrator Login Details](#administrator-login-details) 3. [Technologies Used](#technologies-used) ## Installation Instructions Follow these steps to set up the project locally on your machine. ### Prerequisites 1. Install **XAMPP** (which includes Apache and MySQL). - Download XAMPP from [here](https://www.apachefriends.org/index.html). - Install and start the **Apache** and **MySQL** services from the XAMPP control panel. 2. Download **WordPress**. - Download WordPress from [here](https://wordpress.org/download/). ### Step-by-Step Setup 1. **Place the Project in the `htdocs` Directory:** - Copy the entire `mystore` folder to the `htdocs` directory of your XAMPP installation. - Example: `C:\xampp\htdocs\mystore` 2. **Set Up the Database:** - Open your browser and go to `http://localhost/phpmyadmin`. - Create a new database named `mystore_db`. - Import the database schema if you have a `.sql` backup file for the `mystore_db`. 3. **Configure WordPress:** - Open the `mystore` folder and locate the `wp-config.php` file. - Update the database settings with the following values: ```php define('DB_NAME', 'mystore_db'); // Database Name define('DB_USER', 'root'); // Database User define('DB_PASSWORD', ''); // Database Password (default for XAMPP is empty) define('DB_HOST', 'localhost'); // Database Host ``` 4. **Install the Required WordPress Themes and Plugins:** - **Astra Theme**: Go to the WordPress admin dashboard (`http://localhost/mystore/wp-admin`), and install the Astra theme. - **WooCommerce Plugin**: Install the WooCommerce plugin from the WordPress dashboard to enable the store functionality. 5. **Run the Website:** - Open your browser and go to `http://localhost/mystore`. - The website should now be accessible on your local server. ## Administrator Login Details - **Admin Username**: `admin` - **Admin Password**: `@adia2024@` Use these credentials to log in to the WordPress admin dashboard (`http://localhost/mystore/wp-admin`). ## Technologies Used - **WordPress**: Content Management System (CMS) - **WooCommerce**: E-commerce Plugin for WordPress - **Astra Theme**: Lightweight, customizable WordPress theme - **PHP**: Server-side scripting language - **MySQL**: Database management system - **XAMPP**: Local server environment Feel free to modify and extend the website according to your needs. If you encounter any issues during installation, check the XAMPP logs or WordPress documentation for troubleshooting.
